# Algorithms & Problem-Solving - Level 2 🚀

Welcome to my repository dedicated to the **Algorithms & Problem-Solving (Level 2)** stage of my programming journey. This repository contains the practical applications and final projects developed using **C++**.

## 📌 Projects Included

### 1. Project 1: Rock-Paper-Scissors Game
* **Description:** A console-based game against the computer featuring round management, score tracking, and visual feedback using terminal colors.
* **Concepts Applied:** Enums, Structs, Pass by Reference, Random Number Generation, and Modular Functions.

### 2. Project 2: Math Quiz Game
* **Description:** An interactive math quiz application that generates random arithmetic questions based on customizable difficulty levels and operation types, complete with a final performance evaluation.
* **Concepts Applied:** Complex Structs, Switch Statements, Game Loop Logic, and Dynamic Difficulty Adjustment.

## 🛠️ Key Highlights
* Focus on applying **Clean Code** principles (meaningful naming, modular functions, and readability).
* Solved over 50 logical programming challenges throughout the course to sharpen problem-solving skills.
